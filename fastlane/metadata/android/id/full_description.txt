WordPress untuk Android menempatkan kekuatan penerbitan web dalam saku Anda. Ini adalah pembuat situs web dan masih banyak lagi!

MEMBUAT
- Buatkanlah rumah untuk ide besar Anda di web. WordPress untuk Android adalah pembuat situs web dan pembuat blog.
- Pilih tampilan dan nuansa yang tepat dari berbagai pilihan tema WordPress, lalu sesuaikan dengan foto, warna, dan font sehingga Anda menjadi unik.
- Termasuk didalamnya Tips Mulai Cepat yang memandu Anda melalui dasar-dasar penyiapan untuk membuat situs web baru Anda berhasil.

MENERBITKAN
- Buat pembaruan, cerita, pengumuman esai foto -- apa saja! -- dengan editornya.
- Hidupkan postingan dan halaman Anda dengan foto dan video dari kamera dan album Anda, atau temukan gambar sempurna dengan koleksi fotografi pro gratis dalam aplikasi.
- Simpan ide sebagai draf dan kembali ke sana saat inspirasi Anda kembali, atau jadwalkan posting baru untuk masa mendatang sehingga situs Anda selalu segar dan menarik.
- Tambahkan tag dan kategori untuk membantu pembaca baru menemukan postingan Anda, dan saksikan audiens Anda bertambah.

STATUS
- Periksa statistik situs web Anda secara real time untuk melacak aktivitas di situs Anda.
- Lacak posting dan halaman mana yang mendapatkan aktivitas terbanyak dari waktu ke waktu dengan menjelajahi wawasan harian, mingguan, bulanan, dan tahunan.

PEMBERITAHUAN
- Dapatkan pemberitahuan tentang komentar, suka, dan pengikut baru sehingga Anda dapat melihat orang-orang bereaksi terhadap situs web Anda saat itu terjadi.
- Balas komentar baru saat mereka muncul untuk menjaga percakapan tetap mengalir dan mengakui pembaca Anda.

PEMBACA
- Jelajahi ribuan topik berdasarkan tag, temukan penulis dan organisasi baru, dan ikuti orang-orang yang menarik minat Anda.
- Tunggu postingan yang membuat Anda terpesona dengan fitur Simpan untuk nanti.

BAGIKAN
- Siapkan berbagi otomatis untuk memberi tahu pengikut Anda di media sosial saat Anda menerbitkan pos baru.
- Tambahkan tombol berbagi sosial ke posting Anda sehingga pengunjung Anda dapat membagikannya dengan jaringan mereka, dan biarkan penggemar Anda menjadi duta Anda.

Mengapa WordPress?

Ada banyak layanan blog, pembuat situs web, dan jejaring sosial di luar sana. Mengapa membuat situs web Anda dengan WordPress?

WordPress menguasai sepertiga web. Ini digunakan oleh blog hobi, bisnis dari semua skala, toko online, bahkan situs berita terbesar di internet. Kemungkinannya adalah banyak situs web favorit Anda menggunakan WordPress.

Dengan WordPress, Anda memiliki konten Anda sendiri. Jejaring sosial lain memperlakukan Anda sebagai komoditas, dan menganggap kepemilikan konten yang Anda posting. Tetapi dengan WordPress, apa pun yang Anda terbitkan adalah milik Anda, dan Anda dapat membawanya ke mana pun Anda mau.

Apakah Anda memerlukan pembuat situs web untuk membuat situs web Anda, atau pembuat blog sederhana, WordPress dapat membantu. Ini memberi Anda desain yang indah, fitur canggih, dan kebebasan untuk membangun apa pun yang Anda inginkan.
